# Product Sense

## Product Summary

A local-first creative client for AI-driven video work that keeps ideation, shot planning, assets, prompts, generation, and revision inside one coherent workspace.

## Users

Independent creators and studio teams using a local client like a professional editing tool to plan and iterate on AI video projects.

## Decision Rules

- Keep the workflow coherent from project list to canvas to generation review. Users should feel like they are progressing inside one production environment, not jumping between unrelated tools.
- Favor local trust over convenience. A feature that weakens inspectability, persistence, or offline confidence should clear a very high bar.
- When tradeoffs appear, protect the canvas workflow first, then project and workspace management, then secondary integrations and embellishments.
- Default to reducing interruption cost. Users should be able to reopen the client and continue from the same project state with minimal reconstruction.
- Capture durable product decisions in `docs/product-specs/`.
